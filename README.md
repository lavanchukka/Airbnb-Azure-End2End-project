# Airbnb-Azure-EndtoEnd-project
This is an end to end data engineering project that uses different Azure services and tools. Airbnb Dataset from kaggle is used as data source.
In this project we will perform the following tasks:
Create Azure Free Account
Create a resource group
Create a storage account or Azure Data Lake
Create Azure Data Factory
Create Azure Data Factory Data Ingestion Pipeline
PySpark Data Transformation with Azure Databricks
Data Warehousing with Synapse Analytics-Database, Schema, Views, External tables
Establishing connection between Synapse and Power BI
Deleting the project
